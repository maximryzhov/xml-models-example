Xml Models Example

Это приложение на Django загружает модели динамически из XML-файла. Можно добавить новый объект с помощью формы или редактировать существующий, дважды кликнув по значению поля в таблице. Формы для моделей также генерируются динамически штатными средствами Django. Код приложения доступен на Github. Чтобы увидеть список объектов, выберите модель из списка слева.
