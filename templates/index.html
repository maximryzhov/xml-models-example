Xml Models Example

Это приложение на Django загружает модели динамически из XML-файла. Можно добавить новый объект с помощью формы или редактировать существующий, дважды кликнув по соответствующему значению в таблице. Формы для моделей также генерируются динамически штатными средствами Django. Вы можете зайти в административный интерфейс. Логин: admin, пароль: pass. Код приложения доступен на Github.
