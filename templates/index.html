XML Models Example

Это Django-приложение загружает модели динамически из XML-файла, формы для моделей также генерируются динамически. Чтобы увидеть список объектов, выберите модель из перечня слева. Чтобы создать новый объект, добавьте его с помощью формы внизу страницы. Чтобы редактировать существующий объект, дважды кликните по значению поля в таблице. Загружаемые модели также видимы и доступны для редактирования через панель администрирования. Код приложения доступен на Github.
